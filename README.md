🚀 Space Shooter
- A archade style space shooter built with Godot Engine and GD Script. Pilot your spaceship , destroy incoming meteors , avoid meteors, avoid collisions and survive as long as possible while achieving the highest score.

✨ Features
- Smooth player movement
- Laser shooting mechanics
- Random meteor spawning
- Score tracking
- Sound effects
- Game Over screen
- Collision Detection

🛠️ Built With
Engine: Godot 4.x
Language: GDScript

## Controls

| Key | Action |
|------|--------|
|   ↑   | Move Up |
|   ←   | Move Left |
|   ↓   | Move Down |
|   →   | Move Right |
| Space | Shoot |

## How to Play

1. Launch the game.
2. Move the spaceship using **Arrow Keys**.
3. Press **Space** to shoot.
4. Destroy meteors to survive.
5. Avoid collisions.
6. Try to achieve the highest score.


▶️ How to Run
- Option 1
Download the latest release and run:
SpaceShooter.exe
- Option 2
Open the project using Godot 4.x and press F5.
